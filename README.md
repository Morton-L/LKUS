## LKUS是什么?
一个可以升级内核的脚本

## LKUS有哪些功能？

* 方便快捷的`功能实现`
    *  以root用户执行一行命令即可实现
    *  可选择主线版本或稳定版本
* 回显全`中文`高亮显示,方便阅读
* 代码`全开源`，方便定制自己想要的功能
* 使用`Apache 2.0协议`
* `注释清晰`便于学习

## 如何使用?
